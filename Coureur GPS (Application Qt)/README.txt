Si vous voulez tester l'application {
	Il faut installer Python : www.python.org
	Ne pas oublier de l'installer à la variable d'environnement PATH
	Il faut ensuite redémarrer l'ordinateur

	Il faut ensuite installer Qt Creator : 

	Il faut créer un compte gratuit et si l'installeur vous pose la question vous souhaitez travailler avec la version open source. 
	L'installeur est disponible ici : https://www.qt.io/download-qt-installer 
	Attention le téléchargement est de l'ordre du gigaoctet et peut prendre du temps.

	Lancer l'installeur et remplir les informations
	Sur la page "Sélectionner des composants", il faut cocher MinGW 8.1.0 64 ou 32bits selon votre configuration, dans Qt 5.15.0
	Il faut ensuite également cocher MinGW 8.1.0 64 ou 32bits selon votre configuration, dans Designer and developer tools

	Une fois que tout cela est terminé, vous pouvez lancer le script Python "serveur_gps.py" avec Python
	Quand le serveur est lancé, vous pouvez ouvrir "coureur_gps.pro - Shortcut" avec Qt Creator

	Quand Qt aura chargé les fichiers, vous n'aurez plus qu'a cliquer sur le bouton start vert en bas a gauche de Qt et l'application devrait se lancer
}
Si vous ne voulez pas la tester {
	Des captures d'écran sont disponibles dans le dossier screenshots
